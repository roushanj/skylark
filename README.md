# skylark Maps

  Steps to be followed<br><br>
    1. git clone http://github.com/roushanj/skylark<br>
    2.cd skylark<br>
    3.npm install<br>
    4.Open Terminal change to working directory<br>
    5.nodemon or node server.js or npm start<br>
    6.Goto Browser localhost:3000<br>
